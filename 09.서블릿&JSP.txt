# 서블릿 ?
  was에 의해서 관리되는 객체로 사용자측으로부터 요청이 들어오게 되었을때 작동되게 설계된 객체이다.

# JSP ?
  웹 요청을 처리하기 위한 서블릿을 쉽게 만들어내기 위해서 고안된 언어/기술.
  파일의 위치는 [Java Resource] 쪽이 아니라 [WebContent] 의 하위에 만들면 된다.


  JSP 파일에 기술된 내용은 기본이 출력으로 처리가 된다.
  대신, 자바코드를 용도에 맞게 요소 설정해서 써야된다.
  <%!    %>  : declaration 의 용도 -
               init,destory를 설정해야된다거나, 공용변수나 메서드를 설계하고자 할때
  <%     %>  : scriptlet 의 용도	 -
               요청 처리과정에서 자바를 이용해서 데이터 처리를 하고자 할 때
  <%=    %>  : expression 의 용도 -
               응답 출력과정에서 자바로 처리한 데이터를 출력 하고자 할때

  <%--   --%> : JSP 파일에서만 사용가능한 comment 영역
  
  <%@    %>  : directvie 의 용도 
             WAS가 JSP를 토대로 서블릿을 만들때 참고해야될 설정을 지시하고자 할때
