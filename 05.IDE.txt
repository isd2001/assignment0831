IDE ? (Integrated Development Environment, IDE)

실제 제공되는 객체들을 활용해서 프로그램을 만들어 나가는 연습을 할텐데,

editplus / notepad++  같은 텍스트편집기 로는 번거러움이 너무 많다.
보통 실제 개발을 할때는 IDE 를 사용하는데,
자바 개발에는 Eclipse / netBeans / IntelliJ 같은 것들이 많이 사용된다.

Eclipse 가 가장 기본 IDE로 사용되니까, 이걸로 작업부터 해보자.
